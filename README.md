## Group Webpage and associated documentation

This is the *personal* page of [Pierre Darancet](https://www.anl.gov/cnm/person/pierre-darancet) aiming at documenting his research projects at [Argonne National Laboratory](https://www.anl.gov), [Northwestern](https://www.naise.northwestern.edu/people/) and general projects elsewhere. 
Contents and views expressed here are personal, and in no way are official opinions or representative of Argonne National Laboratory and/or Northwestern University. 

## How to edit:

This site is hosted by github pages. 

Each project has a public or private source and is hosted in a distinct repository. 

For each repository, create a new branch (i.e. not master) called gh-pages. Please be aware everything committed under this branch will be public (regardless of the private/public status of the original repository). The project doc will be hosted under pierretdarancet.github.io/repositoryname/ and can be linked to the original website.
